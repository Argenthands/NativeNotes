# Primera App de React Nativ

## [React Native](https://reactnative.dev/)
### Instalando:

```bash
npm install -g react-native-cli
```

#### Creando la aplicación:

```bash
react-native init [nombre_aplicacion]
```
## [Expo](https://expo.dev/)
### Instalando:

```bash
npm install -g expo-cli
```

#### Creando la aplicación:

```bash
expo init [nombre_aplicacion]
```
